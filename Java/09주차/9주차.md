2. 예외처리 전략에 대해 알려주세요.

- 커스텀 예외를 만들 때 어떤 상황에서 Unchecked를 써야하고 어떤 상황에서 Checked를 써야하나요?

4. 예외처리를 그냥 로직으로 사용하지 않고, Exception을 사용하는 이유는 뭔가요?
